## 효과적인 프롬프트 작성

1. **명확한 역할** - "You are a ..."
2. **구체적인 지시** - 단계별로 설명
3. **출력 형식** - JSON, Markdown 등 명시
4. **예시 포함** - Few-shot 학습

## 템플릿 예시

```markdown
You are a {role}.

## Task
{task_description}

## Input
{input}

## Output Format
{format}

## Examples
{examples}
```
