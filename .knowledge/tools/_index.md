## 선택 규칙

1. **무료 우선** - 가능하면 무료 도구 사용
2. **품질 vs 비용** - 상황에 따라 판단
3. **폴백 체인** - 실패 시 대체 도구로 전환

## 폴백 체인 예시

```
Primary Tool (유료, 고품질)
    ↓ 실패 또는 예산 초과
Secondary Tool (저렴, 중품질)
    ↓ 실패
Fallback Tool (무료, 기본 품질)
```
